JobHunt

A branch from FITS, JobHunt is a job search tracking tracking website built with React, Flask, and MySQL.

About

JobHunt is a branch from another, similar project I created (FITS) that allows users to enter and save information for job postings that they find on various job listing websites (Indeed, LinkedIn, etc.). I built this website because I wanted a tool that I could use to help me keep track of jobs I found while searching for new job opportunities.

JobHunt was created with a lot of the same functionality that FITS already contained, (basic CRUD app, item tracking), which is why it was a branch and not a completely new project. However, over time, the project expanded well past the originally scope of FITS and became its own beautiful thing. JobHunt has a cleaner design, expanded functionality, and more fun JS/Python code that adds flavor to the site.

Originally, JobHunt existed as a full stack Flask app, where Flask handled both the rendering of the web pages as well as the "back end" calls to the MySQL database. However, I started learning React and discovered that I could add some functionality much more easily using React than I could with Flask, so I set out to convert JobHunt to React, first as a learning exercise but eventually migrating and deploying it as a full site. The front end is hosted on Netlify, and the backend (a pared down version of the original Flask app) is still deployed on Heroku.

In JobHunt, a user can add, edit, view, and delete job postings that they find online. JobHunt uses the information that the user enters to create convenient links to information such as the company's office location and the application site. A user can personalize their job postings by listing the experience requirements that they do and do not feel that they meet. They can also rate jobs that they add from 1-5 stars. In the main jobs table, jobs can be filtered by status, with more features to come.
